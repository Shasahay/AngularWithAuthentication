API Calls


Here you can call our API. Just select a user, and a list of applications (often just one application), then click "Get details" and wait maybe a few seconds for the results. It will show the response as a JSON string.

The response from the GetUserInformation API call contains a list of entitlements: these are all the custom entitlements that the user is entitled to. There are several ways a user can have an entitlement, either directly or indirectly:

  • The user has that custom entitlement directly.
  • The user has a standard entitlement which includes that custom entitlement.
  • The user’s company has that custom entitlement directly (account-level entitlement).
  • The user’s company has a standard entitlement directly which includes that custom entitlement.
