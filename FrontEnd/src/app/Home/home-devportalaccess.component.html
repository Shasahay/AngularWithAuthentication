If you can see this, you have developer access to the developer portal. This grants you the capability to:


  • Test Users: Create and manage your own test users in A&E
  • Entitlements: Edit your own entitlements, as well as those of your own test users
  • API Calls: Test the API calls like GetUserInformation and view the response

There is a higher level of access for team leads.


  • Applications: A team lead can register new applications, view the keys for them, etc.
